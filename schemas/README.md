# Data Schemas (v1)

All hub JSON files carry three common fields:

| Field | Type | Notes |
|---|---|---|
| `schema_version` | string | e.g. `"v1"` |
| `generated_at`   | ISO8601 string | producer's local time with offset |
| `source`         | string | producing station id (`crossmart-<x>`) |

## selection.json (Selector → Hub)

```jsonc
{
  "schema_version": "v1",
  "generated_at": "2026-07-11T00:00:00+08:00",
  "source": "crossmart-selector",
  "week": "2026-W28",
  "items": [
    {
      "keyword": "batana oil",
      "country": "US",
      "top_asins": ["B0XXXXX"],
      "score": 87,
      "strategy": "gap",         // gap | volume | margin | trend
      "notes": ""
    }
  ]
}
```

## monitor.json (Monitor → Hub)

- Mirror of `crossmart-monitor/frontend/data/rawData.json`.
- Preserves nested `keywords[].top_asins[]` (with launch_date etc.).

## listing.json (Listing → Hub)

```jsonc
{
  "schema_version": "v1",
  "generated_at": "...",
  "source": "crossmart-listing",
  "drafts": [
    {
      "asin": "B0XXXXX",
      "title": "...",
      "bullets": ["..."],
      "description": "...",
      "generated_at": "...",
      "backlink": null
    }
  ]
}
```

## ops.json (Ops → Hub)

```jsonc
{
  "schema_version": "v1",
  "generated_at": "...",
  "source": "crossmart-ops",
  "date": "2026-07-11",
  "ad_keywords": [
    {"keyword":"batana oil","cvr":0.12,"cpc":0.85,"spend":1200,"orders":54}
  ],
  "traffic_structure": {"organic":0.62,"sp":0.28,"sb":0.06,"sd":0.04}
}
```

```jsonc
  "schema_version": "v1",
  "generated_at": "...",
  "source": "crossmart-simulator",
  "predictions": [
    {
      "asin": "B0XXXXX",
      "predicted_at": "2026-07-11",
      "predicted_monthly_units": 5000,
      "predicted_net_margin": 0.18,
      "actual_units": null,
      "actual_net_margin": null,
      "calibration_status": "pending",   // pending | calibrated | stale
      "error_units_pct": null,
      "error_margin_pct": null
    }
  ],
  "calibration": {
    "last_run_at": null,
    "avg_error_units_pct": null,
    "avg_error_margin_pct": null,
    "sample_size": 0
  }
}
```
